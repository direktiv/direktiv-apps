# SMTP

## Input

A simple go application that sends an email via SMTP client. The following json input can be used to send an email.

```json
{
    "from": "",
    "to": "",
    "subject": "SMTP Email",
    "message": "pog champ",
    "server": "smtp.gmail.com",
    "port": 587,
    "password": ""
}
```

## Output

No output is provided upon success.
